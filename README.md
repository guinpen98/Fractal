# Fractal
## usage
It uses an external library called the [DxLib](https://dxlib.xsrv.jp/).

From the source code folder hierarchy, place the DxLib folder you downloaded in `.\..\..\..\..\Downloads\DxLib_VC3_23\DxLib_VC\プロジェクトに追加すべきファイル_VC用` .
